# OneBook 产品准则

## 1. 审美：赛博禅意 (Cyberpunk Zen)
- **对比**: 暗色氛围背景搭配大胆的字体设计 (Outfit, Inter)。
- **色彩**: 深黑 (#050505)、纯白，以及稀疏使用的霓虹点缀色 (青色、洋红)。
- **毛玻璃 (Glassmorphism)**: UI 面板中使用微妙的背景模糊滤镜。
- **Logo**: 动态蝴蝶 (Project Butterfly) 是核心视觉。

## 2. 交互设计
- **微动画**: 平滑的悬停过渡，书籍封面的淡入效果。
- **专注**: 阅读时 UI 元素应消失或变暗，以最大程度减少干扰。

## 3. 文字风格
- 简洁、专业，但在技术反馈中带着一丝“尼奥 (Neo)”的赛博冷酷感。
